# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project, you agree to abide by its terms.

## Reporting

If you experience or witness unacceptable behavior, please open a [GitHub Discussion](https://github.com/inference4j/inference4j/discussions) marked as private, or contact a maintainer directly through GitHub. All reports will be reviewed and handled confidentially.
